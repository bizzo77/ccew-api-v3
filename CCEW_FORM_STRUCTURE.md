# Official CCEW Form Structure

## Page 1

### Header
- NSW Fair Trading logo
- Title: "Online Certificate Compliance Electrical Work (CCEW)"
- Serial No field (top right)

### Section 1: INSTALLATION ADDRESS (Green header)
- Property Name
- Floor, Unit, *Street Number, &/or, Lot/RMB
- *Street Name, Nearest Cross Street
- *Suburb, *State (NSW), *Post Code
- Pit/Pillar/Pole No., NMI, Meter No., AEMO Metering Provider I.D.

### Section 2: CUSTOMER DETAILS (Green header)
- Checkbox: "Please tick if Customer Address details same as installation details"
- *First Name, *Last Name
- Company Name
- Floor, Unit, *Street Number, &/or, Lot/RMB
- *Street Name, Nearest Cross Street
- *Suburb, *State, *Post Code
- Email, Office No., Mobile No.

### Section 3: INSTALLATION DETAILS (Green header)
- *Type of Installation: Residential, Commercial, Industrial, Rural, Mixed Development (checkboxes)
- *Work carried out:
  - New Work, Installed Meter, Network connection (checkboxes)
  - Addition/alteration to existing, Install Advanced Meter, EV Connection (checkboxes)
  - Re-inspection of non-compliant work, Non-Compliance No. (checkbox + field)
- Special Conditions:
  - Over 100 amps, Hazardous Area, Off Grid Installation (checkboxes)
  - High Voltage, Unmetered Supply, Secondary Power Supply (checkboxes)

## Page 2

### Section 4: *DETAILS OF EQUIPMENT (Green header)
- "Select equipment installed and estimate increase of work affected by the work carried out"
- Table with columns: EQUIPMENT, RATING, NUMBER INSTALLED, PARTICULARS
- Rows: Switchboard, Circuits, Lighting, Socket Outlets, Appliances, Generation, Storage (with checkboxes)

### Section 5: *Meters - Installed (I), Removed (R), Existing (E) (Green header)
- Master/Sub Status - No (N), Master (M), Sub (S)
- Table with columns: I, R, E, Meter No., No. Dials, Master/Sub Status, Wired as Master/Sub, Register No., Reading, Tariff
- Multiple rows for meter entries
- Estimated increase in load A/ph
- * Is increased load within capacity of installation/service mains? Yes/No
- * Is work connected to supply? (pending DSNP Inspection) Yes/No

### Section 6: INSTALLERS LICENSE DETAILS (Green header)
- *First Name, *Last Name
- Floor, Unit, *Street Number, &/or, Lot/RMB
- *Street Name, Nearest Cross Street
- *Suburb, *State, *Post Code
- Email, Office No., Mobile No.
- *Qualified Supervisors No., *Expiry Date, Or, *Contractor's License No., *Expiry Date

## Page 3

### Section 7: *TEST REPORT (Green header)
- "In respect to the test carried out by me on the above mentioned installation, I certify that:"
- 1. I have carried out the test below and that the installation has passed the following requirements:
  - Earthing system integrity
  - Residual current device operational
  - Insulation resistance Mohms
  - Visual check that installation is suitable for connection to supply
  - Polarity
  - Stand-Alone system complies with AS4509
  - Correct current connections
  - Fault loop impedance (if necessary)
- 2. I confirm that I have visually checked that the installation described in this Certificate complies with the relevant Acts, Regulations, Codes and Standards:
- 3. *The test was completed on [date field]

### Section 8: TESTERS LICENSE DETAILS (Green header)
- Checkbox: "Please tick if Testers Lic. details same as Installers Lic. details"
- *First Name, *Last Name
- Floor, Unit, *Street Number, &/or, Lot/RMB
- *Street Name, Nearest Cross Street
- *Suburb, *State, *Post Code
- *Email, Office No., Mobile No.
- *Qualified Supervisors No., *Expiry Date, Or, *Contractor's License No., *Expiry Date
- "In my capacity as the Tester, I certify that the electrical work carried out on the above mentioned property was completed by the nominated electrician"

### Section 9: *SUBMIT CCEW (Green header)
- "Please select the energy provider for where this work has been carried out, to email a copy of this CCEW directly to that provider"
- Dropdown: "Choose an Energy Provider..."
- "Please enter the meter providers email to send a copy of this CCEW directly to that provider"
- Email field
- "Please confirm the owners email address to send a copy of this CCEW directly to the property owner"
- Email field
- "I certify that the information provided in this Certificate Compliance Electrical Work (CCEW) is true and correct."
- Signature field (with note: "Signature is only required when providing as a printed copy")
- SUBMIT button (green)
- Note: "If completing this CCEW electronically, please click the SUBMIT button to generate an email with a copy of the CCEW which you can save and send to the NSW Regulator, Customer, the Service Provider and Meter Provider."

## Colors
- Header sections: Green (#8DB04C or similar)
- Text: Black
- Background: White
- Field boxes: White with gray borders

## Layout
- A4 size
- Margins: approximately 20mm all around
- Green header bars span full width
- Fields have visible borders/boxes
- Checkboxes are small squares
- Tables have visible grid lines
